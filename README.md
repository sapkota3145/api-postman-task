# api-postman-task
API Testing with Postman  


## Overview
This task demonstrates how to work with a REST API using Postman.

## Steps
1. Registered a new user using POST `/users`
2. Logged in using POST `/users/login`
3. Retrieved current user data using GET `/user` with Bearer token

## Tools
- Postman
- REST API
- JSON

## Note
Faced 429 (Too Many Requests) error when sending too many requests quickly.
